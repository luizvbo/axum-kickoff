About

Axum Kickoff provides a solid foundation for building web applications using Axum, Toasty ORM, and other best-in-class Rust crates. It follows the architectural patterns and best practices from crates.io, the official Rust package registry.

Tech Stack

🌐

Axum 0.8

Modern, ergonomic web framework built on Tokio

🗄️

Toasty 0.7

Type-safe ORM with SQLite support

🎨

Askama 0.15

Type-safe, compile-time templating

⚡

Tokio 1.x

Async runtime for Rust

Features

✓ Structured configuration with environment variables
✓ Connection pooling with health checks
✓ Rate limiting (token bucket algorithm)
✓ Request logging and tracing
✓ Comprehensive middleware stack
✓ Real IP extraction
✓ Traffic blocking capabilities
✓ CORS support
✓ GZIP compression
✓ Request timeout handling
✓ Panic recovery
✓ Development/Production environment modes

Getting Started

1

Clone the repository

git clone https://github.com/luizvbo/axum-kickoff.git
2

Set up environment variables

Copy .env.sample to .env

cp .env.sample .env
3

Run the server

cargo run --bin server

Server starts on http://localhost:8888

Project Structure
