This demonstrates simple state updates using HTMX partials. The counter value is updated without page reloads.

How It Works

1. Page loads: HTMX triggers POST to /examples/counter/increment
2. Server returns counter partial HTML with current count
3. User clicks increment/decrement
4. HTMX sends POST with current count value
5. Server calculates new count and returns updated partial
6. HTMX swaps the new partial into #counter-display

Key HTMX Attributes Used

  • → hx-trigger="load": Triggers request when element loads
  • → hx-include: Includes additional form data in request
  • → hx-swap="innerHTML": Replaces content inside target
