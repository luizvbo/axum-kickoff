Contact Form Example

This demonstrates a full-page Askama template with an HTMX form that validates and returns partial HTML on success or error.

How It Works

1. User fills out the form and clicks "Send Message"
2. HTMX sends a POST request to /examples/contact
3. Server validates the input
4. On success: Returns a success partial template
5. On error: Returns an error partial template with validation messages
6. HTMX swaps the response into #form-result

Key Patterns

  • → Full-page template: This HTML file is a complete page
  • → HTMX form: Form submits via AJAX without page reload
  • → Partial responses: Server returns HTML fragments, not full pages
  • → CSRF protection: Token included via htmx-config
  • → Loading indicator: Shows during request processing
